# vuejs-essential

## 项目设置
```
yarn install
```

### 热更新编译运行开发环境
```
yarn serve
```

### 打包和编译生产环境
```
yarn build
```

### 自定义配置
See [Configuration Reference](https://cli.vuejs.org/config/).
